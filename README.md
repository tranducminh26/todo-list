# Dự án To-Do List Full-stack

## 1. Giới thiệu
Ứng dụng quản lý công việc cá nhân với đầy đủ tính năng Đăng nhập, Thêm, Sửa, Xóa task.

## 2. Công nghệ sử dụng
- Backend: Node.js, ExpressJS, MySQL, JWT (Auth).
- Frontend: HTML, CSS, JavaScript (Fetch API).
- Database: MySQL (Railway).

## 3. Cách chạy ứng dụng
1. Clone dự án về máy.
2. Chạy lệnh `npm install` để cài đặt thư viện.
3. Tạo database theo file `database.sql` đính kèm.
4. Cấu hình file `.env` với thông tin database của bạn.
5. Chạy lệnh `node index.js`.
6. Mở `index.html` trên trình duyệt.
